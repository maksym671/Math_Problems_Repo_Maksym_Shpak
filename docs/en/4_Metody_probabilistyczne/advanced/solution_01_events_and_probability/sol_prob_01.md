# Solution

## Task 1

**Given:** Sample space $\Omega = \{\omega_1, \omega_2, \omega_3, \omega_4, \omega_5\}$, events $A = \{\omega_1, \omega_3, \omega_5\}$ and $B = \{\omega_2, \omega_3, \omega_4\}$.

**Find the events:**

**1. $A \cup B$ (union of events)**  
Union contains every element that is in $A$ or in $B$ (or in both):

$$
A \cup B = \{\omega_1, \omega_2, \omega_3, \omega_4, \omega_5\} = \Omega.
$$

**2. $A \cap B$ (intersection of events)**  
Intersection contains only elements that are in both $A$ and $B$:

$$
A \cap B = \{\omega_3\}.
$$

**3. $B \setminus A$ (difference of events)**  
Difference $B \setminus A$ contains elements that are in $B$ but not in $A$:

$$
B \setminus A = \{\omega_2, \omega_4\}.
$$

**4. $A \setminus B$**  
Difference $A \setminus B$ contains elements that are in $A$ but not in $B$:

$$
A \setminus B = \{\omega_1, \omega_5\}.
$$
